### Data Types

* String
    * String : Fixed Length
    * V_String : Var Char
    * V_String FORCE
    * V_WString
    * WString : Wide Char
* Numbers
    * Byte : 0 - 255
    * Int26 : $-(2^{15}) to (2^{15})-1$
    * Int32 : $-(2^{31}) to (2^{31})-1$
    * Int64 : $-(2^{63}) to (2^{63})-1$
    * FixedDecimal : 1234.p
    * Float : +/- (3.4E+/-38)
    * Double : +/- (1.7E+/-308)
* Dates/Times
* Booleans
* Spatial Objects - points, lines , polygon

### Change Data Types

* Preparation > Auto Field
    AutoField takes string values only

* Preparation > Select
